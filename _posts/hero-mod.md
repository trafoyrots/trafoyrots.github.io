삼국지5와 영걸전은 각각 Koei 밑에서 찾는다.
경로 지정은 mod 에디터의 path 디렉터리를 찾아주면 된다.
![][2022-11-30-13-52-06-(6)]

영걸전 경로를 path에 지정하면 아래처럼 장수 이미지가 나온다. 
![][2022-11-30-13-52-06-(7)]

삼국지5 경로를 path에 지정하면 아래처럼 빈 이미지가 나온다.
![][2022-11-30-13-52-06]
여기서 새 이미지를 삽입할 수 있다.
![][2022-11-30-13-52-06-(2)]

새 이미지를 삽입했다.
![][2022-11-30-13-52-06-(4)]

교체된 이미지를 저장한다.
![][2022-11-30-13-52-06-(3)]

저장하면 색상이 다소 변한다. 
이 색상이 그대로 반영되지는 않는다.
![][2022-11-30-13-52-06-(5)]

P(한자).vbs 실행을 종료하고 MOD 에디터로 나가서
XX변환.vbs를 실행한다. 창이 나오면 1번을 누르고 실행한다.

다시 P(한자).vbs 실행하고 영걸전 경로를 지정하면
 1번 장수가 유비에서 제갈량으로 바뀐 결과가 보인다.
![][2022-11-30-13-52-06-(8)]

XX생성.vbs가 있는 디렉터리에 FACEDAT.R3를 복사해두고
XX생성.vbs를 실행한다. 1번을 입력하고 확인을 누른다.
FACEDAT.R3를 영걸전 게임 디렉터리에 복사해서 게임을
실행해보면 이미지가 교체되어 있을 테다.

정리하면

1. 삼국지5의 이미지는 1개 뿐이고 여기에 삽입한 이미지가
새 이미지 파일이 된다.

2. 영걸전의 얼굴(한자어).R3은 256개 이미지가 저장된
데이터 파일이다.

3. XX변환.vbs 는 몇 번째 장수 이미지를 새 이미지로 변환할지 묻고
입력한 번호의 장수 이미지를 교체한다. 
결과 얼굴(한자어).R3의 데이터가 바뀐다.

4. XX생성.vbs는 몇 번째 장수 이미지를 새 이미지로 생성할지 묻고
입력한 번호의 장수 이미지를 교체한다.
결과 FACEDAT.R3의 데이터가 바뀐다.

주의할 점
MOD 에디터 압축파일을 풀고 나서
중국어로 된 파일 명을 변경할 필요가 있다.
중국어 윈도우는 상관없을 테지만 폰트가 
일치하지 않아 문제가 생긴다.
에디터의 각 파일들을 헥스에디터로 읽어들여 살펴보면
프로그램이 읽어들이는 여러 파일 명이 뷁어 혹은 한자어로
보일 텐데 이것을 외부 파일 명과 일치시켜야 한다. 
위에 세 번째 이미지를 보면 파일 이름을 뷁어로 일치시킨 것이 보일 테다.
